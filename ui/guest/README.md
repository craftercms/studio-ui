# Guest App

**Instructions**

1. Run `yarn rollup`
2. add `127.0.0.1 authoring.sample.com preview.sample.com` to /etc/hosts
3. add `HOST=authoring.sample.com` to .env.local 

If you're developing and changing, run `yarn rollup:watch`
